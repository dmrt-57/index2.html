# 🌐 İsa Demirtaş - Veri Merkezi & Ağ Uzmanı Portfolyo Sesi

Bu proje; kalfalık belgelerimi, mesleki eğitim geçmişimi, veri merkezi, ağ yönetimi ve sistem izleme (Zabbix, Proxmox, Keenetic, Ubuntu Server vb.) alanlarındaki yetkinliklerimi sergilediğim kişisel portfolyo web sitemdir.

## 🛠️ Kullanılan Teknolojiler (Built With)
Projenin ön yüz geliştirme aşamasında aşağıdaki teknolojiler kullanılmıştır:
* **HTML5** - Semantik web standartlarına uygun sayfa iskeleti
* **CSS3** - Modern ve şık sayfa tasarımı
* **Git & GitHub** - Sürüm kontrolü ve depo yönetimi

---

## ✨ Özellikler (Features)
* 📱 **Responsive Tasarım:** Mobil, tablet ve masaüstü cihazlarla tam uyumlu yerleşim.
* 📜 **Mesleki Geçmiş:** Bilişim Teknolojileri alanındaki Kalfalık Belgesi ve eğitim süreçlerinin şık bir şekilde listelenmesi.
* ✉️ **İletişim Kanalları:** E-posta ve sosyal medya entegrasyonu.
